# ERC-20 Token Display

The Hedera ecosystem supports both <mark style="color:purple;">**Hedera Token Service (HTS)**</mark> assets, as well as <mark style="color:purple;">**EVM ERC-20**</mark> assets — including bridged assets, like wETH or wBTC via **Stargate (LayerZero)**, as well as liquidity pool receipt tokens issued by **Bonzo Vaults** or **Bonzo Lend**.

How these tokens display in your wallet depends on which wallet you use. MetaMask automatically displays EVM ERC-20 assets but [**HashPack**](https://hashpack.app/) requires you to add them to be displayed manually.

{% hint style="info" %}
**Select the tabbed asset categories below for specific addresses and instructions on how to display EVM ERC-20 assets in HashPack.**
{% endhint %}

{% tabs %}
{% tab title="Bridged EVM Assets" %}
**Assets bridged to Hedera via LayerZero (using the Stargate bridge) arrive on the Hedera network as EVM ERC-20 assets — if you're holding an EVM ERC-20 asset in your account, it must be manually configured to display in HashPack, in order for you to "see" it in your wallet. If using MetaMask, the tokens should automatically appear.**\
\
**HashPack Instructions:**

1. Open HashPack wallet and select an account to view.
2. Directly above (and slightly left) of all your assets on display is a <mark style="color:$success;">**+ Add Token**</mark> button — please click or tap this.
3. You should now see a <mark style="color:$success;">**Or Enter a Token ID**</mark> input field in your wallet.
4. Please input the "EVM Address" of any asset (from the table below) that you'd like visible in your wallet, into the <mark style="color:$success;">**Or Enter a Token ID**</mark> field.
5. Click or tap the <mark style="color:$success;">**Associate Token**</mark> button.

{% hint style="success" %}
Note: Unlike token associations for HTS assets, "associating" an EVM ERC-20 asset in HashPack does not charge a network / transaction fee to your account.
{% endhint %}

{% tab title="Bonzo Lend LP Tokens" %}
**When assets are supplied to or borrowed from the Bonzo Lend protocol,  the user also receives EVM ERC-20 "receipt" tokens that represent the user's supply and debt. These tokens are "invisible" to HashPack wallet and** [**Hashscan**](https://hashscan.io/) **explorer — however, you can make them visible in HashPack wallet, with unique token icons, by following the instructions below.**\
\
**HashPack Instructions:**

1. Open HashPack wallet and select an account to view.
2. Directly above (and slightly left) of all your assets on display is a <mark style="color:$success;">**+ Add Token**</mark> button — please click or tap this.
3. You should now see a <mark style="color:$success;">**Or Enter a Token ID**</mark> input field in your wallet.
4. Please input the "EVM Address" of any asset (from the table below) that you'd like visible in your wallet, into the <mark style="color:$success;">**Or Enter a Token ID**</mark> field.
5. Click or tap the <mark style="color:$success;">**Associate Token**</mark> button.

{% hint style="success" %}
Note: Unlike token associations for HTS assets, "associating" an EVM ERC-20 asset in HashPack does not charge a network / transaction fee to your account.
{% endhint %}
